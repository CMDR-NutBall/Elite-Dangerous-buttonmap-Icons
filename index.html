Icons for Elite Dangerous buttonmaps

These are all the icons that can be used in a custom buttonmap for Elite Dangerous.

For more details on how to make use of these icons in Elite Dangerous, please refer to the documentation that can be found at: EliteCustomButtonNames Repository

Notice the spelling of xb1htsPOVRght. Notice that Right is spelled wrong (Rght no letter i). This is how it is spelled in the Readme file that you can find in your Elite Dangerous install at EDLaunch\Products\elite-dangerous-odyssey-64\ControlSchemes\DeviceButtonMaps

Clicking on the name of the icon will copy it to your clipboard for easy pasting into your own custom buttonmaps
